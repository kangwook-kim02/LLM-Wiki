# Skill: query

## 트리거

ingest / lint 트리거에 해당하지 않는 **모든 질문**에 이 스킬을 적용한다.

- 예시:
  - `"Plan Mode가 뭐야?"`
  - `"Hook과 Loop의 차이가 뭔가요?"`
  - `"Orchestrator 패턴은 언제 써?"`

---

## 절차

### Step 1 — 인덱스 확인
`wiki/index.md` 를 읽어 질문과 관련된 페이지를 파악한다.

### Step 2 — 관련 페이지 읽기
관련성 높은 페이지를 읽는다 (최대 5개).

### Step 3 — 답변 합성
읽은 페이지를 바탕으로 답변을 합성한다.
- 인용 형식: `([[페이지명]] 참고)`
- 위키에 없는 내용이면 "위키에 아직 없는 내용입니다" 명시

### Step 4 — 보존 가치 판단
답변이 새로운 통찰을 담고 있거나 재사용 가치가 높으면:
- `wiki/topics/` 또는 `wiki/concepts/` 에 새 페이지로 저장
- `wiki/index.md` 업데이트

### Step 5 — 로그 기록 (선택)
중요한 질의응답이면 `wiki/log.md` 에 append한다:

```markdown
## [오늘날짜] query | 질문 요약
```

---

## 출력

- 인용 포함 답변
- 참고한 페이지 목록
- (해당 시) 새로 생성된 페이지 경로
