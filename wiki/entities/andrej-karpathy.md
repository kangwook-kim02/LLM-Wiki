# Andrej Karpathy

## 요약
Tesla AI 전 책임자이자 OpenAI 공동 창립자. "Vibe Coding"이라는 용어를 처음 정의하고 LLM Wiki 패턴을 제안한 AI 연구자.

## 주요 기여

- **Vibe Coding 용어 창시**: "AI가 만든 변경을 Accept All로 밀어붙이는 방식"으로 자연어 기반 코드 생성 패러다임을 명명함 (2024.4Q~2025.3Q)
- **LLM Wiki 패턴 제안**: RAG 대신 LLM이 Markdown 위키를 직접 유지·진화시키는 개인 지식 베이스 아키텍처를 [GitHub Gist](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)에 공개
- **소프트웨어 엔지니어링 패러다임 전환 예측**: 자연어/의도 기반 프로그래밍 시대를 조기에 예측

## LLM Wiki 핵심 아이디어

Karpathy가 제안한 LLM Wiki의 핵심:

> "위키는 점점 성장하는 영구적인 산출물이다."

- RAG: 질문마다 원본 문서 재검색 → 지식이 누적되지 않음
- LLM Wiki: LLM이 소스를 읽고 Markdown으로 정리 → 위키가 점점 성장
